About NutriDiary

NutriDiary is a web application that helps users understand and track nutritional information of various fruits. This application is designed to provide useful knowledge for the general public, researchers, and health practitioners.

Feature

Nutri-Checker

Features nutritional value and education about fruits.

Your Food

Save the user's favorite fruit list and perform data management (CRUD).

Nutri-AI

Retrieval-Augmented Generation (RAG) based intelligent features using the Gemini 2 Flash model..

Made With

Flask Tailwind MySQL Pinecone Sentence-Transformer Gemini-2.0-Flash

It is hoped that NutriDiary can be an educational and informative tool that is useful in understanding the content of fruits interactively.
